# 🌐 Capitals - URL Proxy

A modern, feature-rich URL proxy with an intuitive settings interface. Designed for privacy, speed, and ease of use.

## ✨ Features

- **URL Proxying**: Browse any website through the proxy
- **Privacy Controls**: Ad/tracker blocking, JS disabling, user agent spoofing
- **4 Settings Tabs**: General, Privacy, Appearance, Advanced
- **Dark/Light Themes**: Auto-detect or manual selection
- **Settings Export/Import**: Backup your configuration as JSON
- **Quick Shortcuts**: One-click access to popular sites
- **No Backend**: Runs entirely in your browser

## 🚀 Usage

1. Enter a URL or search term
2. Click "GO" or press Enter
3. Configure preferences in Settings (⚙️)

## ⚙️ Settings

- **General**: Search engine, caching, refresh intervals
- **Privacy**: Ad/tracker blocking, JavaScript control
- **Appearance**: Theme, font size, compact mode
- **Advanced**: Custom proxy, CORS bypass, timeout
